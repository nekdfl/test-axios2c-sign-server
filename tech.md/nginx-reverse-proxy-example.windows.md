# Nginx reverse-proxy и Windows

Конфигурация **Nginx** как reverse-proxy к `demo-sign-server` в этом репозитории описана для типичного **Linux**-хоста (пути `/etc/nginx/…`, `systemctl`). На **Windows** нативного сценария нет: используйте **WSL**, **Linux VM** или отдельный сервер и следуйте [`nginx-reverse-proxy-example.ubuntu.md`](nginx-reverse-proxy-example.ubuntu.md).

Сборка и запуск самого Axis2/C на Windows здесь не рассматриваются — см. [`README.ubuntu.md`](../README.ubuntu.md).
